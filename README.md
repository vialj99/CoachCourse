# CoachCourseTrail

Application web de coaching virtuel pour la course à pied et le trail running.

## Fonctionnalités

- Plans d'entraînement personnalisés
- Simulateur d'allure
- Conseils nutritionnels
- Suivi des performances
- Gestion des objectifs
- Profil utilisateur

## Installation

```bash
npm install
npm run dev
